# /plan - VTEX Plan

$ARGUMENTS

---

## Purpose

Create a VTEX-only plan with clear blocks, apps, and integrations.

---

## Behavior

1. **Scope**
   - Pages/blocks to change
   - Apps affected
   - Integrations (Master Data, OMS, Checkout)

2. **Breakdown**
   - Block changes
   - React component updates
   - Backend service changes (if any)

3. **Risks**
   - Site Editor duplication
   - Performance impact
   - Release/rollback

---

## Output Format

```markdown
## VTEX Plan
1. [Block changes]
2. [Component changes]
3. [Integration changes]
4. [Validation steps]
```
