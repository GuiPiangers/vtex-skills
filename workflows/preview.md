# /preview - VTEX Preview

$ARGUMENTS

---

## Purpose

Preview changes in a VTEX workspace.

---

## Behavior

1. **Workspace**
   - Confirm workspace name

2. **Link**
   - Use `vtex link` for local changes

3. **Validate**
   - Check affected pages and devices
   - Confirm Site Editor content renders correctly

---

## Examples

```
/preview new home banner layout
/preview PDP changes
```
