# /status - VTEX Status

$ARGUMENTS

---

## Purpose

Provide current VTEX status: scope, workspace, and app versions.

---

## Behavior

- Confirm account and workspace
- List touched apps/blocks
- Note pending validations (PDP, cart, checkout)

---

## Examples

```
/status current workspace
/status app versions
```
