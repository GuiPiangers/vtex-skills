# /enhance - VTEX Enhance

$ARGUMENTS

---

## Purpose

Incremental improvements within VTEX scope.

---

## Behavior

1. **Identify target**
   - Block tree, component, or integration

2. **Confirm constraints**
   - Site Editor maintenance impact
   - Performance and SEO considerations

3. **Implement minimal change**
   - Prefer block composition over custom code

---

## Examples

```
/enhance home shelf performance
/enhance PDP layout for mobile
/enhance masterdata form UX
```
